# Additional Resources for Learning AI & Humanoid Robotics

This section provides a curated list of external resources to supplement your learning journey in AI and Humanoid Robotics. These resources include official documentation, influential papers, online courses, communities, and open-source projects that can deepen your understanding and provide further avenues for exploration.

## 1. Official Documentation

*   **ROS 2 Documentation**: The official hub for all things Robot Operating System 2. Essential for understanding core concepts, tools, and libraries.
    *   [docs.ros.org](https://docs.ros.org/)
*   **Gazebo Simulation**: Comprehensive guides and tutorials for the Gazebo robot simulator.
    *   [gazebosim.org](https://gazebosim.org/docs)
*   **NVIDIA Isaac Sim**: Official documentation and tutorials for NVIDIA's robotics simulation platform.
    *   [developer.nvidia.com/isaac-sim](https://developer.nvidia.com/isaac-sim)
*   **Docusaurus**: For understanding how this textbook is built and how to contribute to its structure.
    *   [docusaurus.io/docs](https://docusaurus.io/docs)
